<analysis>**original_problem_statement:**
The user wants to build a web application for creating LED light layouts on letters for signage.

**PRODUCT REQUIREMENTS:**
1.  **Input:**
    *   Load any TTF font from the user's local machine.
    *   Input text (e.g., JOHN).
2.  **Layout & Design:**
    *   Apply LEDs in three modes: 'Fill' (grid pattern), 'Border' (outline), or 'Both'.
    *   Use 12mm OD (Outer Diameter) pixels.
    *   Control letter size from 30cm to 300cm.
    *   Control spacing between characters in 'cm'.
    *   Define pixel spacing with separate sliders for border and fill.
    *   Control the number of pixels on the border.
    *   Maintain a 3-5mm safety margin from the letter edge where pixels cannot be placed.
    *   The letter outline should be a thick, visible blue border.
    *   Prominently mark the first and last pixel of every letter.
3.  **Wiring & Connectivity:**
    *   Implement smart wiring to keep all wiring paths within the letter boundaries, avoiding jumps across hollow parts.
    *   Support for T-8000 controllers with 8 selectable ports, each handling 1024 pixels.
    *   Users can place port markers on the canvas and wire them to the start of a letter's pixel sequence.
    *   Each letter must have a 'start' and 'end' node for wiring connections.
    *   Allow users to disconnect the wiring between letters without re-sequencing the pixel numbers.
4.  **Editing & Tools:**
    *   The canvas should support zooming on the selected letter.
    *   Provide a 'Select' tool for rubber-band selection to move or cut groups of pixels.
    *   A 'Break Apart' option to handle letters in a string independently.
    *   A 'Break' wiring feature to disconnect a single pixel.
    *   A 'Wire' tool to manually join two pixels, continuing the flow.
    *   Right-click should add a pixel. A subsequent right-click should act as an 'Escape' key to cancel the last operation.
    *   Implement a 10-step 'Undo' button.
5.  **Export:**
    *   Generate CNC-ready output in  format.
    *   Generate  format specifically for LedEdit software.
    *   The export should include wiring direction and node markings.
6.  **Platform:**
    *   Initially a web app, with a future goal of converting it to a desktop application.

**User's preferred language**: English
